58  * the function will not search for next segment if toStart

66 /*
67  * Convert a Global (segment, X, Y) position into a Local one (segment, toRight, toStart)
68  *
69  * The segment in the Global position is used to start the search of a good segment
70  * in term of toStart value.
71  * The segments are scanned in order to find a toStart value between 0 and the length
72  * of the segment for straights or the arc of the curve.
73  *
74  * The sides parameters is to indicate wether to use the track sides (1) or not (0) in
75  * the toRight computation.
